# My Personal Portfolio

Welcome to the repository for my personal portfolio website! This project serves as my digital home on the web, showcasing my skills, projects, and experiences as a developer. 

You can check out the live site here: [www.sumitshrivastava.me](http://www.sumitshrivastava.me)

## 🚀 About the Project

I built this portfolio to keep a clean, accessible record of what I'm working on and to share my technical journey. It is completely responsive, lightweight, and structured to highlight my projects and skills clearly.

### Tech Stack Used:
* **Frontend:** Semantic HTML5, CSS3 (Custom styling & responsive layouts), Javascript (ES6+)
* **Hosting & Deployment:** Vercel

## 📂 Project Structure

The project structure is kept simple and straightforward for optimal loading performance:
* `index.html` - The main structure of the website.
* `style.css` - Custom UI styling and mobile-responsive media queries.
* `script.js` - Dynamic interactions and frontend logic.
* `/projects` - Assets and code snippets highlighting my work.
